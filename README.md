# CANTO كانتو — موقع تعريفي

موقع تسويقي عربي (RTL) لمقهى مختص + خدمة سيارة (Drive-Thru).

## التفاصيل
- **الاسم:** CANTO كانتو (شعار لاتيني + شعار عربي "كوفي مختص")
- **التصنيف:** مقهى مختص + خدمة سيارة (Drive-Thru)
- **المدينة:** بريدة — حي السلطانة (وعدة فروع)
- **التقييم:** 4.8 على خرائط قوقل (٨٨٤ مراجعة)، ويصل إلى 4.9 في فروع أخرى.

## سبب الاختيار
نشاط حيوي بصور حقيقية جميلة (لاتيه آرت، خدمة سيارة ليلًا، لوحة العلامة)، علامة واضحة الهوية،
وتقييم عالٍ يستحق حضورًا رقميًا أنيق يعكس مستوى القهوة المختصة وسرعة الخدمة.

## فكرة التصميم
اتجاه "قهوة مختصة داكنة وعصرية": خلفية إسبريسو شبه سوداء (#17120E)، لمسات كراميل/عنبري (#C58A3D)،
وكريمي دافئ (#EFE6D8). هيرو داكن بصورة كاملة العرض، حبيبات (grain) خفيفة، شريط متحرك (marquee)،
كشف ناعم عند التمرير، وتكبير الصور عند المرور. خط العناوين Tajawal الثقيل، والنص Cairo،
وشعار "CANTO" بخط Poppins.

## التقنيات
- HTML / CSS / JavaScript **فانيلا** بالكامل — بدون أي مكتبات أو خطوة بناء.
- خطوط Google عبر `<link>` مع `preconnect`.
- IntersectionObserver للكشف عند التمرير (مع fallback يضمن ظهور كل المحتوى).
- بيانات منظمة JSON-LD من نوع `CafeOrCoffeeShop` مع التقييم 4.8/884.

## حالة الـ Backend
**غير مربوط.** نموذج الطلب تجريبي: يحفظ الطلب في `localStorage` ويعرض رسالة نجاح (toast).

> ملاحظة مهمة حول التواصل: رقم **920001555** هو **خط موحّد (920)** ولا يدعم واتساب.
> لذلك تم استبدال زر واتساب العائم بزر **اتصال** + زر **الخريطة**، ولم يُربط النموذج بأي رابط `wa.me`
> ولم يُختلق أي رقم واتساب. زر "احفظ الطلب" يخزّن الطلب محليًا ويوجّه العميل للاتصال على الخط الموحّد لتأكيده.

## مصدر الصور
صور حقيقية للنشاط من ملفه على خرائط قوقل (`assets/img/ct-1…ct-6.jpg`):
- `ct-2`, `ct-3`: خدمة السيارة + لاتيه آرت (هيرو ومعرض)
- `ct-5`: لوحة العلامة المضيئة ليلًا (الموقع/المعرض)
- `ct-6`: كوب على طاولة ترازو بضوء الشمس (القائمة)
- `ct-1`, `ct-4`: أكواب القهوة (المعرض/القائمة)
